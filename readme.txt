Custom Cursor allows you to add and customize animated cursors to your WordPress site. Choose from different styles, colors, and animations.

== Description ==

This plugin lets you enhance the user experience by adding custom cursors. Features include:
- Various cursor styles (circle, triangle, star, etc.).
- Animated cursor effects (e.g., rotation).
- Custom colors and sizes.
- Support for adding your own cursor images.
- Optional admin interface cursor customization.

== Installation ==

1. Download the plugin.
2. Upload the plugin files to the `/wp-content/plugins/custom-cursor` directory, or install via the WordPress Plugin Directory.
3. Activate the plugin through the 'Plugins' screen in WordPress.
4. Go to Settings -> Custom Cursor to configure the plugin.
